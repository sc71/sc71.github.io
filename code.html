True/False Quiz

The True/False Quiz Android app is an app that consists of 10 true or false questions. Each correct answer earns 100 points. After all questions are answered, the user's total score is displayed. This app works in both horizontal and vertical orientations and preserves data upon orientation change.

A Room of Stuff

A Room of Stuff is an interactive room that contains a magic 8 ball and a computer with which you can play Space Invaders.

Restaurants

The Restaurant app uses the online database Backendless. With this app you can create a personal account in which you can record information about restaurants.

Motivational Dog

Motivational Dog takes one image of a shiba inu from the shibe.online API and one quote from the quotesondesign.com API and displays them. This program also uses picasso and retrofit.
